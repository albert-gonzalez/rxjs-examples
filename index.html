Example 6

Observable that emits events from input keyup. It debounce the events 500 ms, map the event to the input value and filter only the values with length greater than 3. A Subject is subscribed to the Observable, and this Subject emits the value of the input when the button is clicked. A subscriber does an AJAX call with the emitted values.
